## Naming convention

- Use lowercase with dashes: `hero-orchard.jpg`
- Formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`
- Recommended width: 1200–2400px for hero, 800–1600px for cards
- Keep files under 500KB when possible (Next.js will optimize)
